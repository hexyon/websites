Websites

Slide Pro

A sleek, feature-rich image slideshow web app. Upload multiple images and control the slideshow with start, stop, shuffle, fullscreen, loop, and progress bar options. Customize slide delay, navigate with arrow keys, and enjoy smooth transitions with a responsive design. Perfect for showcasing photo collections.

HD Photo Filter

HD Photo Filter is a web app that allows users to upload photos, classify them by resolution, and download them according to their preferred size categories. The app provides a user-friendly interface for managing photo resolutions, with options to adjust the minimum resolution for classification and keep track of different resolution categories.

Reader

Reader is a simple web app designed for rich text editing and display. The application features a clean, minimalist design with rounded edges and a user-friendly interface. Users can input text into an editor with a customized toolbar and submit it to display the formatted content below.

Word Theme

A word-themed interactive webpage with a detailed description displayed and a thumbnail image that expands into a full-screen gallery. Navigate through images using arrow keys or on-screen arrows. The design includes responsive elements, smooth transitions, and user-friendly navigation for an engaging experience.

Random Number Generator

A simple web app that generates and copies random 12-digit numbers. It features a button to generate numbers, a display area, and a copy-to-clipboard function. The design includes a responsive layout, and interactive buttons. Ideal for quick and easy random number generation.

Photo Gallery

A minimalist photo gallery web app with a clean design. It features image upload, thumbnail previews, and full-screen viewing with smooth transitions. Navigate through images using arrow keys or on-screen buttons. Perfect for showcasing your photo collections.

Gradient Designer

Gradient Designer is a web tool for creating dynamic gradient backgrounds. Users can generate random gradients, manually select colors, or upload a photo to extract its colors for the gradient. Features include start/stop animations, color code display and copying, and downloading the gradient as an image. Perfect for designers and developers.
